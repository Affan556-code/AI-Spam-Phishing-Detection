# AI-Based Spam and Phishing Message Detection System

## Project Overview
This project uses Machine Learning and Natural Language Processing (NLP) to classify messages as Spam/Phishing or Safe.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- TF-IDF Vectorization
- Logistic Regression
- Google Colab

## Results
- Accuracy Achieved: 96.32%

## Features
- Spam Detection
- Phishing Detection
- Text Classification
- Machine Learning Based Prediction
